# Stock Manager Local

A desktop inventory management app built with Tauri + React + TypeScript.

## ✨ Features
- 📦 Product management (add/edit/delete)
- 📊 Real-time stock tracking
- ⚠️ Low stock alerts
- 📜 Transaction history
- 🔒 100% local - your data stays private

## 🚀 Live Demo
[https://stock-manager-local.builder.kiloapps.io](https://stock-manager-local.builder.kiloapps.io)

## 🛠️ Built With
- Tauri (Rust backend)
- React + TypeScript
- Tailwind CSS
- SQLite
